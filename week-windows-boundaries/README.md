# Week 7 — Windows security boundaries (study notes)

This week is about “Can code run / access the thing at all?” rather than “Can an exploit work?”

## Boundary concepts

- Integrity levels and app containers
- Protected processes and credential isolation
- Script/command execution controls (policy-based)
- Telemetry and detection surface (what defenders see)

## Defensive goal

Understand how these boundaries reduce blast radius and how to validate they’re configured correctly.

## Outcome

You can look at a Windows environment and describe:
- which boundaries are enabled,
- what they protect,
- and which attack paths they reduce.
